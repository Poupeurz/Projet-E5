Le JavaScript c'est quoi ?

JavaScript est un langage de programmation utilisé pour rendre les pages Web interactives. Il peut être utilisé pour manipuler le contenu HTML, modifier le style des éléments, réagir aux événements utilisateur et bien plus encore.

  • Déclaration de variables :En JavaScript, vous pouvez déclarer des variables avec let, const, ou var.

  • Commentaires : Les commentaires en JavaScript peuvent être sur une seule ligne avec // ou sur plusieurs lignes avec /* */.
  • Points-virgules : Les instructions en JavaScript se terminent généralement par un point-virgule (;), bien que cela ne soit pas strictement nécessaire dans tous les cas.
  • Blocs de code : Les blocs de code en JavaScript sont délimités par des accolades { }.
  • Sensibilité à la casse : JavaScript est sensible à la casse, ce qui signifie que variable et Variable sont considérées comme deux variables distinctes.

  • Espacement : L'espacement (ou indentation) est une pratique courante pour rendre le code plus lisible, mais il n'est pas requis par JavaScript.
