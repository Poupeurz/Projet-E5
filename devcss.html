Introduction au css

Le CSS c'est quoi ?

CSS (Cascading Style Sheets) est un langage de feuilles de style utilisé pour décrire la présentation d'un document HTML. Il permet de contrôler l'apparence et la mise en forme des éléments d'une page Web.

Les bases de CSS
  • forum

  • Sélecteurs : Les sélecteurs CSS permettent de cibler les éléments HTML auxquels vous souhaitez appliquer des styles. Par exemple, h1 cible tous les éléments

  • Propriétés et valeurs : Les propriétés CSS définissent les aspects que vous souhaitez styliser, comme la couleur, la taille ou la police du texte. Les valeurs sont les paramètres que vous attribuez à ces propriétés.
  • Déclaration : Une déclaration CSS associe une propriété à une valeur. Par exemple, color: blue; définira la couleur du texte en bleu.
  • img : Balise pour insérer une image.
  • ul et ol : Balises pour créer des listes non ordonnées (à puces) et ordonnées (numérotées), respectivement.

  • li: Élément de liste utilisé à l'intérieur des balises ul ou ol.
